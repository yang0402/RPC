# 简单计数器

一个基于 HTML/CSS/JavaScript 的简单计数器网页应用。

## 功能

- 增加计数（+）
- 减少计数（-）
- 乘以2（×2）
- 除以2（÷2，结果取整）

## 使用方法

1. 下载或克隆此仓库。
2. 在浏览器中打开 `index.html` 文件。
3. 点击按钮进行计数操作。

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6)
